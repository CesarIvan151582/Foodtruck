## Resumen Ejecutivo
Foodtruck es una aplicación web diseñada para gestionar de manera sencilla las operaciones basicas de una negocio tipo food truck, permitiendo visualizar el menu disponible, registrar pedidos y navegar a traves de un menu principal de forma intuitiva

El proyecto esta enfocado en demostrar el uso de buenas prácticas de desarrollo de software, control de versiones, documentación y planeación de mejoras futuras

---
**Problema identificado**

Este negocio pequeño opera de forma manual y poco organizada, lo que provoca errores en pedidos, falta de control y una mala experencia para los cliente

**Solución**
La aplicacion Food Truck centraliza las operaciones básicas en un solo sistemas, facilitando:
- Consulta de menú
- Registro de pedido
- Navegación clara mediante menús
- Base para futuras mejoras y escalabilidad
---
  **Arquitectura**
  La arquitectura del sistema es de tipo monolitica, orientada a una aplicación de escritorio o local:
  - Capa de presentación (interfaz del usuario)
  - Capa lógica de negocio
  - Capa de datos
